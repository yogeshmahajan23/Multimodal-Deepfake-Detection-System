# Multimodal-Deepfake-Detection-System
This paper presents a multimodal deepfake detector using SigLIP for images, Wav2Vec2 for audio, and TensorFlow Lite for video analysis. By combining confidence scores through adaptive fusion, the system achieves 97.3% accuracy and 0.988 AUC-ROC on FaceForensics++, ASVspoof 2021, and DFDC datasets.
